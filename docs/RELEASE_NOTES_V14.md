# RELEASE NOTES V14

## Summary
v14 improves the public homepage experience with a Hebrew-first interface and an English language switcher.

## What changed
- Public homepage now defaults to Hebrew
- English switcher added at the top of the page
- Clearer explanation of the system in Hebrew
- Intake preview form labels switched to Hebrew-first wording
- Login area clarified for admin and customer entry points
- RTL direction support improved on the public homepage

## Why this matters
The public-facing first impression is now much clearer for Hebrew-speaking users while keeping an English option available.
