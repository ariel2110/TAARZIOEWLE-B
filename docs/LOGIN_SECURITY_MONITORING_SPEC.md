# LOGIN SECURITY MONITORING SPEC

## Goal
Give admin clear visibility into public/customer access health.

## Monitor
- recent OTP requests
- recent magic-link requests
- rate limit triggers
- failed login attempts
- delivery preparation attempts
- suspicious repeated requests by phone or IP

## Suggested widgets
- challenges requested in last 24h
- rate-limited events in last 24h
- deliveries prepared in last 24h
- failed customer logins
