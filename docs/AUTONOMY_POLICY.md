# AUTONOMY POLICY

## MVP default
- Level 0: observe/report
- Level 1: safe internal assistance
- Level 2+: approval-gated only

## Never autonomous in MVP
- pricing changes
- billing changes
- destructive customer-impacting actions
- security/auth changes
- broad production content changes

## Allowed autonomously in MVP
- internal reporting
- internal task creation
- alert prioritization
- recommendation drafting
