# CUSTOMER PORTAL DEPTH SPEC

## Purpose
Define the richer customer portal flow introduced in v12.

The customer portal should now support:
- overview
- basic profile/site/account snapshot
- basic edit submissions
- change requests
- support messages
- basic billing/payment visibility
- timeline visibility

- customer overview API
- customer edit submissions

- admin customer ops visibility
