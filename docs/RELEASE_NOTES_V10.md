
# Release Notes v10

## Focus
Turn the operational starter into a more complete control-room style workflow.

## Added in v10
- Real queue actions from UI and API
- Lead qualification and lead-to-business conversion actions
- Move business to draft and mark outreach-ready actions
- Campaign assignment for leads and businesses
- Campaign result snapshot endpoint
- Approval detail view with rationale/evidence/confidence/before-after
- Better CEO Console actions (create task, add decision note)
- Context-aware WhatsApp flow connected to business + draft + city/category
- Better queue metadata with available actions
- Seed data expanded to demonstrate campaigns, approvals, and feedback

## Notes
- v10 is still a starter repo, not production-complete.
- WhatsApp remains admin-assisted.
- Approval actions remain controlled.
- CEO Console is a control room, not a full autonomous operator.
