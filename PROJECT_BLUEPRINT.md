# PROJECT BLUEPRINT

## Summary
This project is an admin-controlled growth and operations platform for:
- importing and qualifying local business leads
- generating premium draft/demo websites
- managing outreach and follow-up workflows
- converting businesses into paying customers
- activating and managing paid sites
- providing a limited customer portal
- measuring performance through analytics
- improving the system through supervised recommendations

## Targeting model
The platform must support prospecting by:
- city
- radius
- category
- score threshold
- website presence
- review count / rating
- campaign / targeting profile

## Major platform areas
1. Lead intake and qualification
2. Draft site generation
3. Outreach and reply tracking
4. Payments and activation
5. Customer portal
6. Analytics and KPI
7. Learning loop and recommendations
8. CEO oversight agent
9. Governance, experiments, and rollback

## MVP
- admin auth
- business CRUD
- draft site CRUD
- basic static generator
- draft preview
- outreach launcher
- payment records
- manual activation
- basic customer portal
- lead import CSV
- KPI overview

## Governance rule
The system may recommend, but meaningful production changes must require approval in MVP.
